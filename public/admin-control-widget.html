💡 Cómo Encender el Servidor

Este widget corre en el navegador y no puede lanzar comandos locales directamente. Para encender el servidor, abre una terminal en la carpeta del proyecto y ejecuta:

Windows:

iniciar.bat

Linux / Mac:

bash iniciar.sh

También puedes iniciar el servidor directamente con:
npm run dev (desarrollo)  /  npm start (producción)
